## Getting Started

```bash
git clone https://github.com/szabolts/adatbalapu24.git

```

### Dev:

```bash
npm install

npm run dev

```

### Build:

```bash
npm run build

npm run start

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
